🔑 How to Get Your AI API Keys

Go Stealth requires an API key to function. Here is how to get them for free or low cost:

Groq (Fastest)
  1. Go to Groq Console
  2. Create an account and click "Create API Key"
  3. Copy and paste into Go Stealth Settings
Google Gemini
  1. Go to Google AI Studio
  2. Click "Get API key" on the left sidebar
  3. Generate a new key for Gemini 1.5 Flash
OpenAI (GPT-4)
  1. Visit OpenAI Platform
  2. Create a new secret key
  3. Ensure you have a few dollars in credit
